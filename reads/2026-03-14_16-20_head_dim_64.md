# Experiment Summary

## Hypothesis
Using HEAD_DIM=64 instead of 128 gives more attention heads (8 vs 4) with the same model dimension, which may improve model capacity without significantly increasing computation.

## Change Made
Changed `HEAD_DIM = 128` to `HEAD_DIM = 64` in train.py. This doubles the number of heads from 4 to 8 while keeping n_embd=512.

## Result
- val_bpb: 0.464691 (previous best: 0.465289) - **0.1% improvement**
- training_seconds: ~900
- peak_vram_mb: 1777.1 (same)
- num_steps: ~533 (same)
- num_params_M: 50.3 (similar)
- n_head: 8 (was 4)

## Expected?
Yes - more heads can capture more patterns without significant overhead.

## Decision
- Keep - new best!
