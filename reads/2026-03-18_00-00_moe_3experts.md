# Experiment Summary

## Hypothesis
MoE (Mixture of Experts) architecture with 3 experts (1 shared + 2 active) could improve model capacity without proportionally increasing compute, leading to better val_bpb.

## Change Made
Replaced MLP with MoE layer:
- 3 experts total
- Top-3 routing (all 3 always active since we need shared + 2 others)
- Each expert is a 2-layer FFN
- Router network to weight expert outputs

## Result
- val_bpb: 0.531863 (baseline: 0.457189)
- peak_vram_mb: 2662.9
- num_params_M: 123.2
- Performance degraded by ~16% relative to baseline

## Expected?
No. MoE was expected to improve capacity but the routing overhead and increased parameter count hurt training efficiency within the 15-minute time budget.

## Decision
- Reject
- MoE adds complexity without val_bpb improvement for this time budget
